# Mobius Home Assistant Add-ons

This repository hosts Home Assistant add-ons related to the EcoTech Mobius ecosystem. Each add-on lives inside its own folder.

## Available add-ons

### `mobius_entities`
Publishes Radion/Vortech telemetry as MQTT sensors for Home Assistant. See `mobius_entities/README.md` for installation and usage instructions.

## Usage
1. In Home Assistant go to **Settings → Add-ons → Add-on Store → ⋮ → Repositories**.
2. Add `https://github.com/justin654/mobius-ha-addon` as a custom add-on repository.
3. Install any add-on listed above, configure it, and start it.
